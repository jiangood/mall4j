# 订单管理

订单管理分用户端和管理端。

## 用户端

用户端订单入口：

```text
yami-shop-api/src/main/java/com/yami/shop/api/controller/MyOrderController.java
```

主要能力：

- 查询我的订单。
- 查询订单详情。
- 用户侧订单操作。

## 管理端

管理端订单入口：

```text
yami-shop-admin/src/main/java/com/yami/shop/admin/controller/OrderController.java
```

主要能力：

- 分页查询订单。
- 查询订单详情。
- 发货。
- 导出待发货订单。
- 导出已销售订单。

## 订单状态

订单状态枚举：

```text
yami-shop-bean/src/main/java/com/yami/shop/bean/enums/OrderStatus.java
```

| 状态值 | 含义 |
| --- | --- |
| `1` | 待付款 |
| `2` | 待发货 |
| `3` | 待收货 |
| `4` | 待评价 |
| `5` | 成功 |
| `6` | 失败或关闭 |

## 定时任务

管理端存在订单定时任务：

```text
yami-shop-admin/src/main/java/com/yami/shop/admin/task/OrderTask.java
```

主要处理：

- 超时未支付订单取消。
- 已发货订单自动确认收货。

XXL-JOB 默认配置在 `application-dev.yml` 和 `application-docker.yml` 中。
