# Mall4j 技术文档

本文档面向项目维护者和二次开发者，目标是让新接手的人：

- 30 分钟跑起后端接口、管理后台、MySQL、Redis。
- 2 小时读懂项目结构、技术框架和权限体系。
- 1 天完成一个简单后台管理功能的二次开发。

## 本地默认端口

| 服务 | 默认端口 | 入口 |
| --- | --- | --- |
| 管理端后端 `yami-shop-admin` | `8085` | `WebApplication` |
| 用户端后端 `yami-shop-api` | `8086` | `ApiApplication` |
| 管理后台 `front-end/mall4v` | `9527` | `pnpm run dev` |
| MySQL | `3306` | 数据库名 `yami_shops` |
| Redis | `6379` | 默认库 `0` |
