# 数据模型

本目录记录核心表结构、字段含义、表关系和业务概念映射。表结构以 `db/yami_shop.sql` 为准。

读完本目录后，你应该能：

- 知道初始化 SQL 里有哪些核心表。
- 按商品、购物车、订单、用户、权限几条主线读表。
- 找到后台菜单和按钮权限对应的数据表。
- 二开新增后台功能时，能设计一张简单业务表，并知道是否需要 `shop_id`、`status`、时间字段。
