# 前后端调用链路

管理后台的典型调用链路如下：

```text
页面 Vue
  -> http.adornUrl('/xxx/yyy')
  -> VITE_APP_BASE_API
  -> 后端 Controller
  -> Service
  -> Mapper
  -> MySQL / Redis
```

## 管理后台请求封装

管理后台请求工具在：

```text
front-end/mall4v/src/utils/http.js
```

接口地址拼接规则：

```text
import.meta.env.VITE_APP_BASE_API + actionName
```

例如：

```text
VITE_APP_BASE_API = 'http://127.0.0.1:8085'
actionName = '/shop/notice/page'
最终请求 = 'http://127.0.0.1:8085/shop/notice/page'
```

## 动态菜单路由

菜单来自后端：

```text
GET /sys/menu/nav
```

前端根据菜单 `url` 加载页面：

```text
菜单 url: shop/notice
页面路径: front-end/mall4v/src/views/modules/shop/notice/index.vue
```

新增管理页面时，菜单 URL 和页面目录必须匹配。
