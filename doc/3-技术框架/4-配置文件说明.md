# 配置文件说明

本地开发主要关注 `dev` 配置。

## 后端环境配置

管理端：

```text
yami-shop-admin/src/main/resources/application.yml
yami-shop-admin/src/main/resources/application-dev.yml
```

用户端：

```text
yami-shop-api/src/main/resources/application.yml
yami-shop-api/src/main/resources/application-dev.yml
```

`application.yml` 默认启用：

```text
spring.profiles.active: dev
```

## 常改配置

| 配置 | 文件 |
| --- | --- |
| 管理端端口 | `yami-shop-admin/src/main/resources/application-dev.yml` |
| 用户端端口 | `yami-shop-api/src/main/resources/application-dev.yml` |
| MySQL 地址、账号、密码 | 两个 `application-dev.yml` |
| Redis 地址和库 | 两个 `application-dev.yml` |
| 文件上传、七牛云、短信 | `yami-shop-common/src/main/resources/shop.properties` |
| 用户端支付回调域名等 | `yami-shop-api/src/main/resources/api.properties` |
| 管理端 worker 配置 | `yami-shop-admin/src/main/resources/admin.properties` |

## 前端配置

管理后台：

```text
front-end/mall4v/.env.development
```

小程序：

```text
front-end/mall4m/utils/config.js
```

uni-app：

```text
front-end/mall4uni/.env.development
```
