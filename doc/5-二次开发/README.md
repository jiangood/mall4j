# 二次开发

本目录帮助新维护者完成第一个简单功能。

读完本目录后，你应该能：

- 找到一个后台管理功能对应的后端、前端、菜单和权限位置。
- 照着现有简单模块新增一个后台 CRUD 功能。
- 判断“菜单不显示、按钮不显示、接口 401、页面空白”这类问题应该查哪里。

建议阅读顺序：

1. [二开总览](1-二开总览.md)
2. [新增后台管理功能-帮助中心文章](2-新增后台管理功能-帮助中心文章.md)
3. [二开文件位置速查](3-二开文件位置速查.md)
4. [后台 CRUD 接口模板](4-后台CRUD接口模板.md)
5. [新增接口](5-新增接口.md)
6. [新增前端页面](6-新增前端页面.md)
7. [新增菜单与权限](7-新增菜单与权限.md)
8. [权限配置速查](8-权限配置速查.md)
9. [二开常见卡点](9-二开常见卡点.md)
10. [学习检查题-含答案](10-学习检查题-含答案.md)

第一阶段推荐示例是“帮助中心文章管理”，因为它覆盖表、后端接口、前端页面、菜单和权限，但不碰订单交易主链路。
