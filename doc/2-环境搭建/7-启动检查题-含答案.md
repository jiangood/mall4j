# 启动检查题-含答案

这些问题不是考试，是帮你在启动失败时快速定位。

## 环境

问题：后端要求哪个 JDK 大版本？

答案：JDK `17`。

问题：管理后台要求哪个 Node.js 版本？

答案：以 `front-end/mall4v/package.json` 为准，当前是 `^20.19.0` 或 `>=22.12.0`。

问题：管理后台为什么用 `pnpm run dev` 这种显式脚本命令？

答案：当前文档统一使用显式脚本命令 `pnpm run dev`，对应 `package.json` 中的 `dev` script。

## 数据库和 Redis

问题：本地数据库名是什么？

答案：`yami_shops`。

问题：初始化 SQL 在哪里？

答案：`db/yami_shop.sql`。

问题：Redis 默认端口是多少？

答案：`6379`。

## 后端

问题：管理端后端启动类是什么？

答案：`yami-shop-admin/src/main/java/com/yami/shop/admin/WebApplication.java`。

问题：用户端后端启动类是什么？

答案：`yami-shop-api/src/main/java/com/yami/shop/api/ApiApplication.java`。

问题：只跑管理后台时，必须启动哪个后端？

答案：必须启动 `yami-shop-admin`，默认端口 `8085`。

## 前端

问题：管理后台本地页面端口是多少？

答案：当前是 `9527`。

问题：管理后台接口地址配置在哪里？

答案：`front-end/mall4v/.env.development`。

问题：管理后台默认应连接哪个接口地址？

答案：`http://127.0.0.1:8085`。

## 登录

问题：初始化账号密码是什么？

答案：`admin / 123456`。

问题：登录接口 401，第一步查什么？

答案：先查管理后台是否连到了 `8085`，再查后端是否启动、Token 是否过期、权限数据是否正常。

问题：验证码或登录接口异常，可能是连错端口吗？

答案：可能。管理后台连到用户端 `8086` 会出现接口不匹配或权限异常。
