# 环境要求

依赖版本以仓库内配置文件为准。

## 后端

| 工具 | 要求 |
| --- | --- |
| JDK | `17` |
| Maven | 能构建 Maven 多模块工程即可 |
| MySQL | `5.7+` |
| Redis | `4.0+`，本地默认端口 `6379` |
| XXL-JOB | 可选，当前依赖版本 `2.4.2` |

后端关键依赖版本见根目录 `pom.xml`：

- Spring Boot `4.0.3`
- MyBatis-Plus `3.5.16`
- Sa-Token `1.44.0`
- Redisson `4.3.0`
- Knife4j `4.5.0`

## 管理后台

管理后台位于 `front-end/mall4v`，以 `package.json` 为准：

| 工具 | 要求 |
| --- | --- |
| Node.js | `^20.19.0` 或 `>=22.12.0` |
| pnpm | `>=7` |

常用命令：

```powershell
cd front-end\mall4v
pnpm i
pnpm run dev
```

## 完成后应该确认

- `java -version` 显示 JDK `17`。
- MySQL 可以连接本地 `3306`。
- Redis 可以连接本地 `6379`。
- `pnpm -v` 满足 `>=7`。
- Node.js 版本满足 `front-end/mall4v/package.json` 的 `engines.node`。
