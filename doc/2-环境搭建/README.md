# 环境搭建

本目录目标是跑通最小可运行闭环：

```text
MySQL + Redis + yami-shop-admin + front-end/mall4v
```

读完本目录后，你应该能：

- 启动管理端后端 `8085`。
- 启动管理后台 `9527`。
- 登录后台管理页面。
- 判断启动失败时应该先查数据库、Redis、端口还是接口地址。

建议阅读顺序：

1. [30 分钟启动路线](1-30分钟启动路线.md)
2. [环境要求](2-环境要求.md)
3. [数据库初始化](3-数据库初始化.md)
4. [后端本地启动](4-后端本地启动.md)
5. [管理后台本地启动](5-管理后台本地启动.md)
6. [前端接口地址配置](6-前端接口地址配置.md)
7. [启动检查题-含答案](7-启动检查题-含答案.md)

用户端接口、小程序、uni-app 可在管理后台跑通后再启动。
