# 前端接口地址配置

不同前端连接不同后端。

| 前端 | 配置文件 | 默认接口 |
| --- | --- | --- |
| 管理后台 `mall4v` | `front-end/mall4v/.env.development` | `http://127.0.0.1:8085` |
| 小程序 `mall4m` | `front-end/mall4m/utils/config.js` | `http://127.0.0.1:8086` |
| uni-app `mall4uni` | `front-end/mall4uni/.env.development` | `http://127.0.0.1:8086` |

## 管理后台

```text
VITE_APP_BASE_API = 'http://127.0.0.1:8085'
```

对应后端：

```text
yami-shop-admin
```

## 小程序和 uni-app

```text
front-end/mall4m/utils/config.js
front-end/mall4uni/.env.development
```

对应后端：

```text
yami-shop-api
```

如果出现 `401`、验证码异常、接口不存在，优先检查前端是否连错端口。
