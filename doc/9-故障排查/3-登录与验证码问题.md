# 登录与验证码问题

## 后台登录地址打不开

先确认：

- 管理端后端 `yami-shop-admin` 已启动。
- 管理后台 `front-end/mall4v` 已启动。
- 前端 `VITE_APP_BASE_API` 指向 `http://127.0.0.1:8085`。

## 验证码接口异常

常见原因是管理后台连错了接口端口。

管理后台应该连接：

```text
http://127.0.0.1:8085
```

不要连接用户端：

```text
http://127.0.0.1:8086
```

## 默认后台账号

初始化 SQL 中默认账号：

```text
admin / 123456
```

如果账号登录失败，检查：

- `tz_sys_user` 是否导入。
- `tz_sys_role` 是否导入。
- `tz_sys_role_menu` 是否导入。
- `tz_sys_menu` 是否导入。
