# 启动失败

## 后端启动失败

优先检查：

- JDK 是否为 `17`。
- MySQL 是否启动。
- Redis 是否启动。
- `application-dev.yml` 中数据库账号密码是否正确。
- `yami_shops` 数据库是否已导入 `db/yami_shop.sql`。
- 端口 `8085` 或 `8086` 是否被占用。

## 管理后台启动失败

管理后台位于：

```text
front-end/mall4v
```

确认 Node 和 pnpm 版本满足 `package.json`：

```text
node: ^20.19.0 || >=22.12.0
pnpm: >=7
```

启动命令：

```powershell
pnpm i
pnpm run dev
```

## Docker 启动失败

当前 `docker-compose.yml` 引用了 `db/Dockerfile`，但当前仓库没有该文件。直接 `docker compose up -d` 可能失败。

处理方式见：

```text
doc/8-部署运维/Docker部署.md
```
