# 后端模块说明

后端是 Maven 多模块工程，父工程为根目录 `pom.xml`。

| 模块 | 职责 |
| --- | --- |
| `yami-shop-admin` | 管理端接口，启动类为 `WebApplication`，本地默认端口 `8085`。 |
| `yami-shop-api` | 用户端接口，启动类为 `ApiApplication`，本地默认端口 `8086`。 |
| `yami-shop-sys` | 系统管理能力，包括菜单、角色、日志、配置等。 |
| `yami-shop-service` | 业务 Service、Mapper、Mapper XML，承载主要业务逻辑和数据访问。 |
| `yami-shop-bean` | 数据对象，包括实体、DTO、VO 等。 |
| `yami-shop-common` | 通用能力，包括统一响应、异常、工具类、注解、配置属性等。 |
| `yami-shop-security` | 认证、Token、权限判断、登录用户上下文。 |

## 两个后端入口

管理后台调用 `yami-shop-admin`：

```text
front-end/mall4v -> http://127.0.0.1:8085
```

小程序、H5、uni-app 调用 `yami-shop-api`：

```text
front-end/mall4m 或 front-end/mall4uni -> http://127.0.0.1:8086
```

## 二开时文件通常怎么放

新增一个后台管理功能，通常会涉及：

```text
yami-shop-bean/src/main/java/com/yami/shop/bean/model/
yami-shop-service/src/main/java/com/yami/shop/dao/
yami-shop-service/src/main/java/com/yami/shop/service/
yami-shop-service/src/main/resources/mapper/
yami-shop-admin/src/main/java/com/yami/shop/admin/controller/
front-end/mall4v/src/views/modules/
front-end/mall4v/src/crud/
db/yami_shop.sql
```
