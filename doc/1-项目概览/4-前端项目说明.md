# 前端项目说明

`front-end` 目录下有三个前端项目。

| 项目 | 说明 | 默认接口 |
| --- | --- | --- |
| `mall4v` | Vue3 管理后台，二开管理功能时优先看这里。 | `http://127.0.0.1:8085` |
| `mall4m` | 原生微信小程序端。 | `http://127.0.0.1:8086` |
| `mall4uni` | uni-app 多端前端。 | `http://127.0.0.1:8086` |

## 管理后台

管理后台位于 `front-end/mall4v`，技术栈以 `package.json` 为准，当前主要包括：

- Vue3
- Vite
- Element Plus
- AVUE
- Pinia
- Vue Router
- Axios

菜单路由由后端 `/sys/menu/nav` 返回，前端在 `src/router/index.js` 中动态加载：

```text
菜单 url: shop/notice
页面文件: src/views/modules/shop/notice/index.vue
```

所以新增后台页面时，菜单 URL 要和页面目录保持一致。

## 小程序和 uni-app

对维护和二开来说，先掌握它们的接口地址配置：

- `front-end/mall4m/utils/config.js` 修改 `domain`
- `front-end/mall4uni/.env.*` 修改 `VITE_APP_BASE_API`

小程序和 uni-app 的源码结构可以按具体二开需求再深入；当前技术文档先保证读者知道接口地址在哪里配置。
