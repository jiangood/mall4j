# 项目介绍

Mall4j 开源版是一个 Java B2C 单商户商城系统，采用前后端分离架构。

当前仓库包含后端接口、管理后台前端、小程序前端、uni-app 前端、数据库脚本和部署配置。第一阶段接手项目时，建议先关注最小可运行闭环：

- 管理端后端：`yami-shop-admin`
- 用户端后端：`yami-shop-api`
- 管理后台：`front-end/mall4v`
- 数据库：MySQL
- 缓存：Redis

## 适合谁

- 想学习 Java 商城系统实现的人。
- 想基于开源版做二次开发的人。
- 需要快速评估 Mall4j 技术结构的人。
- 接手维护 Mall4j 开源版的人。

## 先理解哪些能力

先看这些主干能力：

- 商品、分类、规格、SKU
- 购物车、确认订单、提交订单、支付
- 用户、地址、订单管理
- 后台菜单、角色、权限
- 文件上传、系统日志、统一异常、分页

营销、供应链、多商户、SaaS、跨境等能力不是开源版第一阶段阅读重点。
