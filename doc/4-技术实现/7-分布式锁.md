# 分布式锁

项目提供基于 Redisson 的分布式锁注解。

核心文件：

```text
yami-shop-common/src/main/java/com/yami/shop/common/annotation/RedisLock.java
yami-shop-common/src/main/java/com/yami/shop/common/aspect/RedisLockAspect.java
```

## 使用方式

在方法上添加：

```java
@RedisLock(lockName = "demo", key = "#param.id")
```

锁的 Redis key 规则：

```text
redisson_lock:{lockName}:{SpEL解析结果}
```

## 注解参数

| 参数 | 说明 |
| --- | --- |
| `lockName` | 锁名称。 |
| `key` | 支持 SpEL 表达式。 |
| `expire` | 锁过期时间，默认 `5000`。 |
| `timeUnit` | 时间单位，默认毫秒。 |

## 当前使用情况

当前代码中提供了注解和切面，但业务代码里暂未发现实际 `@RedisLock` 使用点。需要加锁时应先确认业务场景确实存在并发写冲突。

## 二开建议

适合加锁的场景：

- 库存扣减
- 重复提交
- 同一资源并发状态流转

不适合加锁的场景：

- 普通查询
- 无共享资源写入
- 前端按钮防抖即可解决的问题
