# 系统日志

系统日志用于记录后台关键操作，例如新增、修改、删除。

核心文件：

```text
yami-shop-common/src/main/java/com/yami/shop/common/annotation/SysLog.java
yami-shop-sys/src/main/java/com/yami/shop/sys/aspect/SysLogAspect.java
yami-shop-sys/src/main/java/com/yami/shop/sys/controller/SysLogController.java
```

日志数据表：

```text
tz_sys_log
```

## 使用方式

在需要记录操作日志的方法上加：

```java
@SysLog("保存菜单")
```

现有代码常用于：

- 新增
- 修改
- 删除
- 修改密码
- 菜单和角色维护

## 记录内容

切面会记录：

- 操作描述
- 类名和方法名
- 第一个方法参数
- 请求 IP
- 当前后台用户名
- 执行耗时
- 创建时间

## 二开建议

新增后台管理功能时，写操作建议加 `@SysLog`。查询类接口通常不需要记录。

注意当前切面读取 `args[0]` 作为请求参数，给无参方法加 `@SysLog` 可能不合适。
